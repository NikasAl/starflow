# Star Flow Command
Стратегическая игра в реальном времени — захватывайте планеты, отправляя потоки кораблей.

## Проекты

### starflow-pwa/ — HTML5 Canvas 2D (основной)
Игра на TypeScript + Canvas 2D + Vite. Работает в браузере и как PWA на Android.

```bash
cd starflow-pwa
npm install
npm run dev      # dev server → localhost:3000
npm run build    # production → dist/
```

### starflow-godot/ — Godot 4.5 3D (на паузе)
Оригинальный 3D-проект на GDScript. Требует Godot 4.5+. На данный момент неработоспособен из-за проблем совместимости Godot 4.5 с автозагрузкой скриптов.

## Как играть
- **Клик по своей планете** — выбрать (подсветится)
- **Клик по чужой/нейтральной** — отправить корабли
- **ПКМ + перетаскивание** — перемещение камеры
- **Колёсико мыши** — масштаб
- **Двойной клик** — центрировать на планете

## Стек технологий (PWA)
- TypeScript + Canvas 2D API
- Vite (сборка)
- PWA (установка на Android через Chrome)
