# Star Flow — Audio Prep Tool

Визуальный инструмент для подготовки звуковых файлов игры Star Flow.
Позволяет загрузить MP3-треки, найти в них подходящие фрагменты, разметить маркерами и экспортировать в OGG через ffmpeg.

## Запуск

```bash
# Просто откройте в браузере (Chrome/Firefox/Edge)
xdg-open tools/audio-prep/index.html

# Или через локальный сервер (рекомендуется для стабильной работы Web Audio)
cd tools/audio-prep
python3 -m http.server 8080
# Откройте http://localhost:8080
```

## Рабочий процесс

### 1. Загрузить список целевых звуков

Нажмите **Load README** и выберите файл `starflow-3d/public/audio/README.md`.
Инструмент парсит дерево файлов и строит список всех нужных звуков (SFX и музыка).

### 2. Загрузить исходные MP3-треки

Нажмите **Load Tracks** и выберите один или несколько MP3-файлов.
Треки появятся в левой панели. Можно добавлять треки по мере необходимости.

### 3. Разметка фрагментов

1. **Выберите трек** в левой панели — отобразится волновая форма
2. **Выберите целевой звук** в правой панели (подсветится)
3. **Выделите фрагмент** перетаскиванием мыши на волновой форме
4. В нижней панели покажутся: начало, конец, длительность, примерный размер OGG
5. Нажмите **Assign (A)** или клавишу `A` — фрагмент привяжется к звуку
6. Инструмент автоматически перейдёт к следующему неназначенному звуку

### 4. Навигация

| Действие | Управление |
|---|---|
| Выделить регион | Перетаскивание мыши |
| Прокрутка (панорама) | `Ctrl` + перетаскивание |
| Масштаб | Колёсико мыши или слайдер |
| Масштаб по выбору | Кнопка **Sel** |
| Масштаб на весь трек | Кнопка **All** |
| Воспроизвести регион | `Space` или кнопка **Play Region** |
| Назначить фрагмент | `A` или кнопка **Assign** |
| Удалить назначение | `Delete` / правый клик на звуке |
| Прыжок к региону | Двойной клик на назначенном звуке |
| Отменить выделение | `Escape` |
| Сохранить проект | `Ctrl+S` |

### 5. Настройка качества OGG

В левой панели внизу:

- **SFX** — качество Vorbis (рекомендуется q=3, примерно 112 kbps, mono 44.1kHz)
- **Music** — качество Vorbis (рекомендуется q=5, примерно 160 kbps, stereo 44.1kHz)

При изменении качества пересчитываются все размеры назначенных фрагментов.

### 6. Сохранение проекта

Нажмите **Save Project** — сохраняется JSON-файл со всеми привязками.
Можно загрузить позже через **Open Project** и продолжить работу.

### 7. Экспорт

Нажмите **Generate Script** — скачается shell-скрипт `export-audio.sh`.

Для нарезки и конвертации:

```bash
# Переместите MP3-треки в одну папку
mkdir -p /tmp/starflow-tracks
cp track1.mp3 track2.mp3 /tmp/starflow-tracks/

# Запустите скрипт
INPUT_DIR=/tmp/starflow-tracks OUTPUT_DIR=starflow-3d/public/audio bash export-audio.sh

# Или просто (треки в текущей директории):
INPUT_DIR=. OUTPUT_DIR=starflow-3d/public/audio bash export-audio.sh
```

Скрипт автоматически:
- Нарежет фрагменты по маркерам
- Конвертирует в OGG Vorbis с выбранным качеством
- SFX → mono 44.1kHz, Music → stereo 44.1kHz
- Создаст нужную структуру каталогов

## Структура каталога

```
tools/audio-prep/
├── index.html          # Сам инструмент (самодостаточный HTML)
└── README.md           # Этот файл
```

Инструмент **не** попадает в сборку игры — он находится за пределами `starflow-3d/`
и не обрабатывается Vite/TypeScript.

## Требования

- Браузер с поддержкой Web Audio API (Chrome, Firefox, Edge)
- ffmpeg (для экспорта, уже установлен в системе)
- Python 3 (опционально, для локального HTTP-сервера)
